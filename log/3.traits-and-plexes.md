## Singleton plex

```kodiak
import kodiak.collection.mutable
object checksum-accumulator =
  private val cache = mutable.map.empty[:text, :int64]
  fn calculate(s :text) :int64 =
    if cache.contains(s) then
      cache[s]
    else
      val acc = checksum-accumulator {}
      for c = s do
        acc.add((c >> 8).to-byte)
        acc.add(c.to-byte)
      val cs = acc.checksum
      set cache += { @ s => cs }
      cs`
```
